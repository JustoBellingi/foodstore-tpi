# FoodStore - Trabajo Práctico Integrador

Trabajo Práctico Integrador de la Tecnicatura Universitaria en Programación.

## Integrantes

- Justo Bellingi

## Estructura del proyecto

```
foodstore-tpi/
│
├── foodstore-backend/
└── foodstore-frontend/
```

## Backend

Desarrollado en:

- Java 17
- Gradle
- JPA
- Hibernate


## Frontend

Desarrollado con:

- HTML
- CSS
- TypeScript

### Funcionalidades backend

- Menu de Categorías
- Menu de Productos
- Menu de Usuarios
- Menu de Pedidos
- Reportes de productos por categoría
- JPA
- Hibernate

### Funcionalidades

- Visualización de productos
- Gestión del carrito
- Administración de productos
- Navegación entre pantallas
- Integración con el backend

## Cómo ejecutar

### Backend

cd foodstore-backend
gradle run

### Frontend

cd foodstore-frontend
npm run dev
